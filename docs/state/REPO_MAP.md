# Repo Map

- `/` — repo root and entry points.
- `PowerShell/` — endpoint scripts and reusable functions.
- `PowerShell/Functions/` — shared PowerShell functions.
- `PowerShell/Tools/` — standalone tools.
- `prompts/` — prompt cards and templates.
- `prompts/powershell/` — PowerShell prompt library.
- `prompts/powershell/endpoint/` — endpoint troubleshooting prompt cards.
- `prompts/powershell/ad/` — empty; intended for Active Directory prompt cards.
- `prompts/powershell/intune/` — empty; intended for Intune prompt cards.
- `prompts/powershell/tickets-kb/` — empty; intended for ticket/KB prompt cards.
- `docs/` — standards, workflow, and repo state.
- `docs/state/` — environment and workflow notes.
- `docs/mcp/` — MCP guidance.
- `samples/` — sanitized sample inputs/outputs.
- `samples/sanitization/` — sample data for PII sanitization.
- `scripts/` — batch tasks and workflow notes.

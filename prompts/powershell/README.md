# PowerShell Prompt Library

This folder holds reusable “Prompt Cards” for generating safe, consistent PowerShell scripts and runbooks.

## Rules
- No internal hostnames/domains/ticket IDs in examples.
- Prefer safe/read-only actions first.
- Include validation + rollback notes.
- Keep outputs copy/paste-ready for IT Ops tickets/KBs.

## How to use
Pick a card, fill in Inputs, paste into Claude/ChatGPT, and save the resulting script under `PowerShell/`.
